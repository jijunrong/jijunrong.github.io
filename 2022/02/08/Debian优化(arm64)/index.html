下载引导
cd /boot/efi/EFI &&wget wget https://boot.netboot.xyz/ipxe/netboot.xyz-arm64.efi

！！！系统安装完后必做！！！
修改密码（不会显示输入的密码，输入完了按 enter 回车即可）
passwd root

sudo apt-get install openssh-server
修改相关配置文件，允许使用 root 用户登录

sudo sed -i ‘s/^#?PermitRootLogin./PermitRootLogin yes/g’ /etc/ssh/sshd_config;
sudo sed -i ‘s/^#?PasswordAuthentication./PasswordAuthentication yes/g’ /etc/ssh/sshd_config;

重启 sshd 服务
sudo service sshd restart

5.开启bbr

echo “net.core.default_qdisc=fq” >> /etc/sysctl.conf

7.YouTube-dl及aria2安装配置
apt install youtube-dl

apt-get install aria2

aria2配置

创建下载目录
mkdir /home/aria2
chmod 777 /home/aria2

创建配置文件
mkdir /etc/aria2
touch /etc/aria2/aria2.session
chmod 777 /etc/aria2/aria2.session

测试是否安装成功
aria2c –conf-path=/etc/aria2/aria2.conf

显示如下：
[NOTICE] IPv4 RPC：正在监听 TCP 端口 6800
[NOTICE] IPv6 RPC：正在监听 TCP 端口 6800
[NOTICE] 已成功将当前会话序列化到“/etc/aria2/aria2.session”

ctrl+c可停止运行

创建开机启动

nano /etc/systemd/system/aria2.service

输入内容如下：

[Unit]
Description=aria2

[Service]
User=root
Group=root
ExecStart=/usr/bin/aria2c –conf-path=/etc/aria2/aria2.conf
Restart=on-abort

[Install]
WantedBy=multi-user.target

开机自启查看进程状态及开始关闭
systemctl enable aria2
systemctl start aria2
systemctl status aria2
systemctl stop aria2

配置文件
mkdir /etc/aria2
touch /etc/aria2/aria2.session
chmod 777 /etc/aria2/aria2.session

cd /BOX && sudo mv aria2.conf /etc/aria2/ && mv aria2.service /etc/systemd/system/

9.卸载多余内核

查看当前的内核
uname -sr

查看已安装的内核debian/Ubuntu命令：
dpkg -l | grep linux-image | awk ‘{print$2}’

一键删除未使用的内核debian/Ubuntu命令：
apt -y remove –purge $(dpkg -l | grep linux-image | awk ‘{print$2}’ | grep -v $(uname -r))

重新检测debian/Ubuntu命令：
dpkg -l | grep linux-image | awk ‘{print$2}’

更新引导系统并重启debian/Ubuntu命令：
update-grub
reboot

10.安装xrdp

apt install xrdp

系统一键升级
apt full-upgrade

11.卸载多余软件
apt

12.设置ipv4优先，并禁用ipv6

设置ipv4优先
sed -i “s/#?precedence ::ffff:0:0/96 100/precedence ::ffff:0:0/96 100/“ /etc/gai.conf
恢复默认
sed -i “s/^precedence ::ffff:0:0/96 100/#precedence ::ffff:0:0/96 100/“ /etc/gai.conf

查看VPS的IPv6信息
输入命令: ip -6 addr show scope global
或者 curl ipv6.ip.sb

禁用VPS的IPv6命令:
sysctl -w net.ipv6.conf.all.disable_ipv6=1; sysctl -w net.ipv6.conf.default.disable_ipv6=1

启用VPS的IPv6命令:
sysctl -w net.ipv6.conf.all.disable_ipv6=0; sysctl -w net.ipv6.conf.default.disable_ipv6=0

重新载入sysctl配置(两条命令作用相似)
sysctl –system reload sysctl

13.查看系统基本情况代码

查看版本当前操作系统内核信息
uname -a

查看当前操作系统版本信息
sudo cat /proc/version

查看版本当前操作系统
cat /etc/issue

查看版本当前操作系统发行版信息。
cat /etc/os-release

查看cpu相关信息，包括型号、主频、内核信息等
cat /proc/cpuinfo

Debian查看版本说明当前CPU运行在32bit模式下， 但不代表CPU不支持64bit
getconf LONG_BIT

查看系统简要信息
lsb_release -a

ssh登陆界面美化

1、/etc/issue 本地登陆显示的信息，本地登录前
2、/etc/issue.net 网络登陆显示的信息，登录后显示，需要由 sshd 配置
3、/etc/motd 常用于通告信息，如计划关机时间的警告等，登陆后的提示信息
4、/etc/update-motd.d/10-uname 登陆提示第一行，可做命令！
5、/etc/ssh/sshd_config #PrintLastLog no
